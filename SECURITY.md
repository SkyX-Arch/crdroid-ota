# Security Policy

## Supported Versions

Security updates are provided only for the latest ROM builds.

Older builds are not actively maintained.

## Reporting a Vulnerability

If you discover a potential security issue, please do NOT open a public issue.

Instead, report it privately via:

- GitHub Security Advisories (preferred)
- or direct contact with the maintainer

## Scope

This is a custom ROM project maintained as a hobby.  
While security is taken seriously, this project does not guarantee enterprise-level security auditing or formal security certification.

## Response Time

I will try to respond as soon as possible in my free time.  
However, response times may vary depending on availability.

## Notes

Some reported issues may be classified as:
- upstream Android behavior
- device-specific limitations
- or non-security related bugs
