## Overview

The Xiaomi 12T (codename: *plato*) is a mid-range smartphone from Xiaomi, released in October 2022.

This repository provides **unofficial crDroid Android builds** for this device.

The project is maintained as a hobby in free time.  
There is no guarantee of stability, completeness, or long-term support.

---

## Features

- crDroid Android 16 base
- User builds
- Signed with release keys
- OTA updates supported via built-in updater
- Up-to-date crDroid sources

## Device Specifications

| Spec | Details |
|------|---------|
| SoC | MediaTek Dimensity 8100 Ultra (MT6895) |
| CPU | Octa-core (4x Cortex-A78 @ 2.85 GHz & 4x Cortex-A55 @ 2.0 GHz) |
| GPU | Mali-G610 MC6 |
| Memory | 8GB LPDDR5 |
| Storage | 128/256GB UFS 3.1 |
| Battery | 5000 mAh |
| Display | 6.67" AMOLED, 1220×2712, 120Hz |
| Rear Camera | 108 MP + 8 MP + 2 MP |
| Front Camera | 20 MP |
