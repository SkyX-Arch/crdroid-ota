# Contributing Guide

Thank you for taking the time to contribute to this project.

This ROM is maintained as a hobby project in my free time. Contributions, feedback, and bug reports are welcome, but please follow the guidelines below to keep everything organized.

---

## 🐛 Bug Reports

Before opening a bug report:

- Make sure the issue is reproducible
- Check if it already exists in the Issues section
- Ensure you are using a clean flash (if required)
- Confirm you are on the latest build

When reporting a bug, please include:

- Detailed steps to reproduce the issue
- Expected behavior
- Actual behavior
- Logs (logcat, dmesg, or relevant logs)
- ROM build date

⚠️ Issues without logs or clear steps may be closed.

---

## 💡 Feature Requests

Feature requests are welcome, but:

- Not all requests will be implemented
- Features must align with ROM stability and device limitations
- Development time is limited

Please provide:

- Clear description of the feature
- Reason why it is useful
- Possible implementation ideas (if any)

---

## 📦 General Rules

- Be respectful and constructive
- Do not spam issues with duplicates
- Do not report issues from modified systems unless reproducible on clean flash
- Do not expect instant fixes or ETAs

---

## ⚙️ Development Notes

- This is an unofficial device build
- Some limitations may come from upstream sources or proprietary components
- Device-specific issues may take time to investigate

---

## 🙏 Important Note

This project is maintained in my spare time as a hobby.

While all feedback is appreciated, priorities may change and not every request or bug can be addressed immediately.

---

## 💬 Communication

For issues and suggestions, use GitHub Issues only.

Please avoid private messages for bug reports unless explicitly requested.
